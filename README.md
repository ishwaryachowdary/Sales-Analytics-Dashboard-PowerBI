# Sales-Analytics-Dashboard-PowerBI
Interactive Sales Analytics Dashboard  built using Power BI and Excel


## Project Overview
This project is an interactive Sales Analytics Dashboard developed using Microsoft Power BI and Excel. It provides insights into sales performance using KPIs and interactive visualizations.
## Dashboard Preview
![Sales Analytics Dashboard](<img width="1920" height="1020" alt="image" src="https://github.com/user-attachments/assets/282b84c2-214a-4d3e-9586-44681814fd11" />

## Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity KPI
- Sales Trend Analysis
- Category-wise Sales
- Region-wise Sales
- Product-wise Sales Table
- Interactive Slicers

## Tools Used
- Microsoft Power BI
- Microsoft Excel

## Dataset
The dataset contains:
- Order ID
- Order Date
- Customer Name
- Product
- Category
- Quantity
- Unit Price
- Sales
- Profit
- Region

## Dashboard
The dashboard helps users analyze sales trends, product performance, and regional sales through interactive charts.
